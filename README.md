# Skills

Каталог моих skill-ов для Codex.

## Доступные skill-ы

### research-digest-params

Параметризуемый skill для обзора свежих научных работ и технических материалов.

Что принимает:
- ключевые слова;
- количество работ на выходе;
- период обзора.

Пример использования:

```text
@research-digest-params RIS, 5 работ, 26 год
```
